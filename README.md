# Mainframe Search README

Search a mainframe library thro job running program ISRSUPC.

## Features

- [x] Search the mainframe for a string
- [ ] Filter the search library elements to reduce search job execution time
- [ ] Select the library from the Zowe Explorer
- [x] Show the outDD result of the job in vscode
- [ ] Export the search results in json format
- [ ] Export the search results in outDD format
- [ ] Open a element from the search results
- [ ] Remove the dependency from Zowe CLI


> Note: The extension allows for multi filter selection. Separate every filter with a coma.

## Requirements

- Zowe CLI

## Extension Settings

This extension contributes the following settings:

* `zSearch.JobCard`: Job card definition for the search job.
